# 🎓 Student Score Management System (Mini Project)

A Python-based console application designed to manage student academic records efficiently using procedural programming techniques. This project serves as a practical application of core programming concepts such as modularity, file handling, and data structures.

---

## 📌 Project Overview
This system operates in a CLI (Command Line Interface) environment through a menu-driven interface. It allows users to perform CRUD operations, perform statistical analysis, and ensure data persistence using both TXT and JSON formats.

**Selected Topic:** Topic 3 – Student Score Management (Quản lý điểm sinh viên)

### 🎯 Key Objectives
- **Data Management:** Create, view, search, and manage student information (ID, Name, Score).
- **Analysis:** Provide sorting and grading statistics.
- **Persistence:** Implementation of file I/O for long-term data storage.
- **Principles:** Apply procedural programming and robust input validation.
